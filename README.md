# AI-Content-Creator
You can easily create any content with a few minutes.

# AI Content Studio is a web-based AI-powered content generator that creates platform-specific content (blogs, scripts, social media posts) using Google's Gemini AI model.

# Key Features:
Feature                   Description
Multi-Platform            YouTube, LinkedIn, Instagram/TikTok, Blog, Medium
Tone Control              Professional, Casual, Educational, Persuasive, Urgent
Content Types             Blog Posts, Video Scripts, News, Social Captions, Technical Articles
DOCX Export               Download generated content as formatted Word document
Dark Theme                Modern, sleek dark UI design
